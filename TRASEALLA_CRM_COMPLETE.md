## Testing Guide

### Prerequisites

1. **Node.js** v18+ installed
2. **MySQL** 8.0 running
3. **Git** for version control

### Step 1: Database Setup

```bash
# Login to MySQL
mysql -u root -p

# Create database
CREATE DATABASE trasealla_crm;

# Create user (optional)
CREATE USER 'crm_user'@'localhost' IDENTIFIED BY 'your_password';
GRANT ALL PRIVILEGES ON trasealla_crm.* TO 'crm_user'@'localhost';
FLUSH PRIVILEGES;
```

### Step 2: Backend Setup

```bash
# Navigate to backend
cd /Users/usama/Desktop/trasealla/crm-backend

# Install dependencies
npm install

# Configure environment (create .env file)
cat > .env << EOF
PORT=4000
DB_HOST=localhost
DB_PORT=3306
DB_USER=root
DB_PASSWORD=RootPassword123!
DB_NAME=trasealla_crm
JWT_SECRET=your_secret_key_here
FRONTEND_URL=http://localhost:5173
OPENAI_API_KEY=your_openai_key (optional)
EOF

# Start backend
npm run dev
```

### Step 3: Frontend Setup

```bash
# Navigate to frontend
cd /Users/usama/Desktop/trasealla/crm-frontend

# Install dependencies
npm install

# Start frontend
npm run dev
```

## API Documentation

### Base URL

### Authentication
All endpoints (except /auth/*) require JWT token in header:
```
Authorization: Bearer <token>
```

### Endpoints

| Method | Endpoint | Description |
|--------|----------|-------------|
| POST | /auth/login | User login |
| POST | /auth/register | Register new user |
| GET | /auth/session | Get current session |
| GET | /accounts | List accounts |

## Deployment Guide

### Production Deployment

#### Option 1: Traditional VPS

```bash
# On server
git clone <repo>
cd crm-backend
npm install --production
npm run build
pm2 start dist/index.js --name crm-backend

cd ../crm-frontend
npm install
npm run build
# Serve with nginx
```

#### Option 2: Docker

```dockerfile
# Backend Dockerfile
FROM node:18-alpine
WORKDIR /app
COPY package*.json ./
RUN npm install --production
COPY . .
EXPOSE 4000
CMD ["npm", "start"]
```

```yaml
# docker-compose.yml
version: '3.8'
services:
  backend:
    build: ./crm-backend
    ports:
      - "4000:4000"
    environment:
      - DB_HOST=mysql
    depends_on:
      - mysql
  
  frontend:
    build: ./crm-frontend
    ports:
      - "80:80"
  
  mysql:
    image: mysql:8.0
    environment:
      MYSQL_ROOT_PASSWORD: rootpassword
      MYSQL_DATABASE: trasealla_crm
    volumes:
      - mysql_data:/var/lib/mysql

volumes:
  mysql_data:
```
